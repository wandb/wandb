# Security Policy

We're no longer patching versions of the library <= the 0.9.x series.  

| Version | Supported          |
| ------- | ------------------ |
| 0.10.x  | :white_check_mark: |
| <0.10.x | :x:                |

## Reporting a Vulnerability

Please report all vulnerabilites to security@wandb.com.
